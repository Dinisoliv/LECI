Notas importantes!


O programa usa "requests" para aceder a resursos web.

python3 -m pip install requests

Linux:

sudo apt install python3-requests



O programa usa "unicodedata" para a ordenação dos dados.

Para instalar o módulo usa o seguinte comando:

pip install unidecode


Linux:

sudo apt-get install python3-unidecode



Para melhor visualizar a tabela com a informação das atrações,
é recomendado ter a janela do terminal em "fullscreen".
